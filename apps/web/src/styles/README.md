# Styles

Layered CSS for the clinical-trust design system. Imported from `app/globals.css`:

1. `tokens.css` — CSS variables (color, type, space, shadow, safe-area)
2. `base.css` — reset, document, typography, `.page`, a11y utilities
3. `components.css` — shared primitives (header, button, form, table, toast…)
4. `pages.css` — landing, flow, not-found, responsive overrides

Prefer React components in `src/components/` over inventing new class recipes.
